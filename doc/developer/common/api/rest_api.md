## REST API

This API is **decprecated**, please switch to new [JSON:API](json_api.md).

If you need to refer to the documentation of the read-only legacy REST API, you can find it in the [git history](https://github.com/hitobito/hitobito/blob/b3531268671396689158c04e85ba220782e501a8/doc/developer/common/api/rest_api.md).

### Migration to the new JSON:API

The following table shows the equivalent endpoints in the new JSON:API for each of the old API endpoints.

| Old API endpoint                                    | Function                                                                     | New JSON:API endpoint                                                                                                                                                       |
|-----------------------------------------------------|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `/groups/:id`                                       | Group details                                                                | `/api/groups/:id`                                                                                                                                                           |
| `/groups/:id/people`                                | People of a certain group                                                    | `/api/roles?include=person&filter[group_id]=:id`                                                                                                                            |
| `/groups/:group_id/people/:id`                      | Person details                                                               | `/api/people/:id`                                                                                                                                                           |
| `/groups/:group_id/events`                          | Events of a certain group                                                    | `/api/events?filter[group_id]=:group_id`                                                                                                                                    |
| `/groups/:group_id/events/:id`                      | Event details                                                                | `/api/events/:id`                                                                                                                                                           |
| `/groups/:group_id/events/:id/participations`       | Participations of certain event                                              | Not yet implemented, see [#3726](https://github.com/hitobito/hitobito/issues/3726)                                                                                          |
| `/groups/:group_id/events/:id/participations/:id`   | Participation details                                                        | Not yet implemented, please open an [issue](https://github.com/hitobito/hitobito/issues/new?template=feature.md&title=API:%20Teilnehmer-Details) if you need this endpoint. |
| `/groups/:group_id/events/course`                   | Courses of a certain group                                                   | `/api/events?filter[group_id]=:group_id&filter[type]=Event::Course`                                                                                                         |
| `/groups/:group_id/invoices`                        | Invoices of a certain group                                                  | `/api/invoices?filter[group_id]=:group_id`                                                                                                                                  |
| `/groups/:group_id/invoices/:id`                    | Invoice details                                                              | `/api/invoices/:id`                                                                                                                                                         |
| `/groups/:group_id/mailing_lists`                   | Mailing lists of a certain group                                             | `/api/mailing_lists?filter[group_id]=:group_id` (will become available with [#3613](https://github.com/hitobito/hitobito/pull/3613))                                        |
| `/groups/:group_id/mailing_lists/:id`               | Mailing list details and subscribers (actual people subscribed to the list)  | `/api/mailing_lists/:id` (will become available with [#3613](https://github.com/hitobito/hitobito/pull/3613))                                                               |
| `/groups/:group_id/mailing_lists/:id/subscriptions` | Details on the groups / roles, events and single people which are subscribed | Not yet implemented, please open an [issue](https://github.com/hitobito/hitobito/issues/new?template=feature.md&title=API:%20Abo-Subscriptions) if you need this endpoint.  |
